# Blog

本博客基于 [Astro](https://astro.build) 和 [Markdoc](https://markdoc.dev) 构建。
主题来源于 [astro-theme-docs](https://astro.new/latest/getting-started)，进行了二次开发。

## Feature✨

- **黑夜模式**：所有主题都内置了浅色/深色模式。
- **移动端适配**：响应式设计，所有设备都能快速加载。
- **md文档渲染**：使用 Markdoc 渲染文档。
- **SEO**：所有页面都设置了 SEO。
